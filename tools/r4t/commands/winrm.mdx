# winrm

Windows Remote Management (WinRM) operations.

---

## Usage

```
r4t winrm [flags]
```

> **Status**: Base command registered. Subcommands are not yet implemented.

WinRM support is planned for authentication testing and remote command execution (PSRemoting). When implemented, it will use the global authentication flags against targets on ports 5985 (HTTP) and 5986 (HTTPS).
