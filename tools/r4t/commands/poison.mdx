# poison

Network poisoning attacks.

---

## Usage

```text
r4t poison [flags]
```

> **Status**: Base command registered. Subcommands are not yet implemented.

Poisoning support is planned for LLMNR, NBT-NS, and mDNS name resolution poisoning — the same class of attacks performed by tools like Responder. These attacks intercept broadcast name resolution requests and return attacker-controlled responses to capture Net-NTLMv2 hashes or redirect traffic.
