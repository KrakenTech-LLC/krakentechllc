# tftp

TFTP (Trivial File Transfer Protocol) operations.

---

## Usage

```text
r4t tftp [flags]
```

> **Status**: Base command registered. Subcommands are not yet implemented.

TFTP support is planned for file retrieval from TFTP servers. TFTP is unauthenticated and commonly used for network device configuration backups — misconfigurations can expose sensitive files like router configurations containing credentials.
